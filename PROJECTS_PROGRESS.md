## 🆕 新项目策划

### 4. 《时能魔君》**策划完成**
- **类型**：低武江湖 / 权谋智斗 / 群像
- **状态**：核心设定完成，待创建工作空间
- **参考经典**：
  - 《庆余年》：权谋智斗、朝堂博弈、理性决策
  - 《琅琊榜》：复仇布局、智斗复仇、身份伪装
  - 《赘婿》：商业帝国、利益至上、理性经营
  - 《雪中悍刀行》：群像精彩、江湖+朝堂
  - 《魔道祖师》：正邪对立、身份转变

- **主角**：林不语
  - **身份**：魔教"血煞宗"副教主，穿越的冷静理智者
  - **性格**：
    - 冷静机智，在任何情况下都能保持理智
    - 利益至上，一切决策基于利益最大化
    - 适应规则，在弱肉强食的世界里不谈善恶，只谈利益
  - **金手指**：**时能系统**
    - 能量来源：自然积累（1单位/天）
    - 消耗机制：指数级消耗（凡人1单位/招，宗师10000单位/招）
    - 核心困境：能量不够，必须靠宝物战斗

- **宝物收集系统**：
  - 免费宝物：不耗能量（凡兵、丹药、普通武学）
  - 低耗宝物：1单位/次（名兵、稀有护具）
  - 中耗宝物：10单位/次（法器、秘宝）
  - 高耗宝物：100单位/次（神兵、神器）
  - **主角策略**：收集大量免费和低耗宝物，减少能量消耗

- **世界观**：
  ```
  北方：魔域（魔教总部）
  中央：朝廷（大明京城）
  东方：江南（正道聚集地）
  西方：江北（朝廷控制地）
  南方：草原（游牧部落）
  ```

- **三大势力**：

  **1. 魔教（血煞宗）**
  - 教主：血魔尊（活了几百年的怪物）
  - 四大护法：主角是南方护法
  - 八大分堂：主角管辖南方分堂
  - 十二魔将：魔教顶尖战力
  - 文化：崇尚实力、内斗严重、利益至上

  **2. 朝廷（大明）**
  - 皇帝：明武帝（雄才大略，老谋深算）
  - 六部：吏、户、礼、兵、刑、工
  - 特务机构：锦衣卫、东厂
  - 禁军：朝廷军队
  - 武林盟：朝廷控制的正道联盟
  - 文化：集权统治、利益至上、权谋斗争

  **3. 江湖（正道联盟）**
  - 武林盟主：君子剑柳长青
  - 五大门派：少林、武当、峨眉、昆仑、崆峒

## 💡 核心经验总结

1. **大纲迭代的重要性**：
   - 第1轮通常只能达到B级（7-7.5分）
   - 第2-3轮可以提升到A级（8-8.5分）
   - 第4-5轮才能达到S级（9.5分以上）
   - 建议：每个项目至少3-5轮迭代

2. **核心设定的创新**：
   - 《禁忌收容》："动情即死" + "废料喂食"
   - 《万剑灵匣》："因果化解循环" + "触剑观忆"
   - 《时能魔君》："时能系统" + "宝物收集"
   - 建议：每本书至少1-2个核心创新点

3. **群像刻画的重要性**：
   - 每个角色都要有成长弧光
   - 8-10个主要角色为宜
   - 每个角色都要有创伤和执念

4. **参考经典的重要性**：
   - 不要完全照搬，但要学习其优点
   - 学习《庆余年》的权谋智斗
   - 学习《琅琊榜》的复仇布局
   - 学习《雪中悍刀行》的群像刻画
   - 学习《魔道祖师》的正邪对立
